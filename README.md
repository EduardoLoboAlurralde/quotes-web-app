# 📦 test-next

Template base para iniciar proyectos con **Next.js 15**, **TypeScript**, **ESLint + Prettier** y **Jest + Testing Library**.  
Incluye configuración lista para desarrollo moderno usando **Turbopack**.

---

## 🚀 Requisitos

- Node.js 22
---

## 🛠️ Instalación

```bash
npm install
```

---

## ▶️ Comandos principales

### Desarrollo
Levanta el servidor de desarrollo con **Turbopack**:
```bash
npm run dev
```

Opcional: usar Webpack en lugar de Turbopack:
```bash
npm run dev:webpack
```

### Build
Compila la aplicación para producción:
```bash
npm run build
```

### Lint
Ejecuta ESLint y Prettier:
```bash
npm run lint
```

### Tests
Corre los tests con **Jest + Testing Library**:
```bash
npm test
```
